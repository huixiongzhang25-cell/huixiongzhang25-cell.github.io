# 框架介绍

express是node.js的框架

## 使用该框架

1.安装node.js

进入node.js的官网下载Node.js

官网地址：
(https://nodejs.org)

2.将node.js添加到系统环境变量

 - 目的：能让你在其他的终端中运行node.js相关的命令
 
 3.判断是否成功安装node.js
 
  - 在终端中输入命令
  
    ```cmd
    node -v
    ```
- 该命令用于检查node.js是否已经安装，并检查node.js的版本号

### 安装express框架


>使用npm包管理工具进行安装

- **全局安装**

> 全局安装意味着安装后你可以在任何目录下使用express框架

```cmd
npm install express -g
```

   - install 安装 
   - express 框架名
   - -g  grobal表示全局安装


- **为某个项目安装express框架**

> 相当于在某个文件夹里安装express框架

   1.新建一个文件夹，在该文件夹里打开终端，输入命令

```
npm init -y
```

 - init 初始化命令，
 - -y yes,表示确定，直接为你安装

>运行成功后，npm 会为你创建一个package.json的配置文件，该文件会记录你下载的框架

![[Pasted image 20260331201639.png]]

2.在项目文件夹里安装express框架

   输入命令:
   ```cmd
npm install express
   ```

- 安装成功后会新增**node_modules**和**package-lock.json**

- node_modules文件夹包含了相关框架的代码

- packahe-lock.json文件则更加细致的记录项目的有关信息

如图所示：
![[Pasted image 20260331202812.png]]


> 我们在vscode打开package.json文件夹，可以看到配置文件中多了一个**dependencies**属性，该属性记录了我们这个项目的依赖，即安装的库会记录在这个属性里

同样，在终端输入：

# 在代码中调用express框架

1.新建一个js文件，将express 和 cors导入

![[Pasted image 20260331204942.png|416]]


2.初始化
![[Pasted image 20260331210611.png|347]]
- app=express()

  - 可以理解为app就是服务器本体，后续所有配置都要挂载到这个app对象上
- app.use(cors())
   
   - 用于开启跨域资源共享，cors()不传参默认允许所有域名访问，传参则运行相应的域名访问

- app.use(express.json()) 

   - 解析前端的JSON格式的请求体，当你前端发送{name:"zhang"}的数据
