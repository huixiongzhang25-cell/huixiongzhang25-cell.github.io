# 连接mysql服务器


命令 **mysql -u root -p**

### 命令含义：

- -u user 表示当前你的**用户名**
- root 表示以超级管理员的身份登录
- -p password 表示当前你的密码，密码默认是**123456**

#### 创建数据库

**不加条件创建数据库**

```
CREATE DATABASE <database_name>; --创建一个数据库
```


 **也可以增加条件语句用于条件判断**
 

```
CREATE DATABASE IF NOT EXISTS <databse_name>; --不存在则创建
```


#### 删除数据库


**删除数据库**


```
	DROP DATABASE <database_name>;
```


**存在删除**

```
    DROP DATABASE IF EXISTS <database_name>;
```

# 修改数据表的字段

### 增加数据库的字段

1.不指定位置添加字段

```
ALTER TABLE <table_name>
ADD COLUMN <字段名> 字段类型[约束条件];
```

2.指定位置添加字段

```
ALTER TABLE users
ADD COLUMN phone VARCHAR(11)NULL
COMMNET "手机号" AFTER username;
```

#                          数据的查找(查)

### 查找所有列

```
SELECT * FROM table_name
WHERE student_name="张三";
```

### 查找特定列

```
SELECT username,id FROM table_name
WHERE username="万一丁";
```


### 模糊匹配查找(LIKE)

```
SELECT * FROM table_name
WHERE student_name LIKE "张"

```


### 通配符查找

**%通配符**(表示零个或多个字符)

```
SELECT * FROM table_name
WHERE student_name LIKE "张%"
```

**表示一个字符**

```
SELECT * FROM table_name
WHERE student_name LIKE "_张%"
```

### 合并关键字(UNION)

#### 选取列，并合并

```
SELECT student_name FROM student
UNION
SELECT student_city FROM student
ORDER BYs tudent_city;
```
