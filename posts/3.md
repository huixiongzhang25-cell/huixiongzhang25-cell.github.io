# python脚本获取必应图片
![**今日必应图片**](https://img.wemd.app/1773721215459_ykwwyi.jpg)
### 背景介绍
- 必应是微软旗下的**搜索引擎**，它每天都会提供精选*图片*:smile: 
- 必应官方提供了相应的**接口**，该接口返回的数据提供了图片的一些信息，如标题，**图片url链接**等
-  我们通过访问**图片url链接**能够直接在浏览器上看到图片
### 脚本希望实现的目标
- **自动化**获取图片
- 保存到本地
### 配置信息 ：
- 脚本使用**python语言**
- 需要的python第三方库：**requests**
- 编辑器：**vscode**
 ---

>必应官方提供JSON接口：
  ```
  https://cn.bing.com/HPImageArchive.aspx?format=js&idx=0&n=1&mkt=zh-CN

 ```
 
 ### 接口url查询参数说明:
 | 参数名 | 含义 | 常用取值  |
| :--- | :--- | :--- |
| **format** | 返回的数据格式 | `js` (JSON), `xml` |
| **idx** | 起始天数 **（相对于今天**） | `0` (今天), `1` (昨天) |
| **n** | 获取**图片的数量** | `1(一张)`  |
| **mkt** | 地区 (影响图片内容) | `zh-CN` (中国), `en-US` (美国) |

 ### 完整代码示例
 ~~~
 #导入requests模块
import requests
#官方提供的接口地址，接口返回的数据包含图片的相关信息
api_url='https://cn.bing.com/HPImageArchive.aspx'
#设置查询参数
query={"format":"js",
    "idx":0,
    "n":1,
    "mkt":"zh-CN"
    }
#发起网络请求，获取响应数据
res=requests.get(url=api_url,params=query).json()
title=res["images"][0]["title"]
url=res["images"][0]["url"]
print("获取的标题为："+title)
full_url="https://cn.bing.com"+url
print("获取的图片地址为："+full_url)
#content获取响应结果的二进制内容
image=requests.get(full_url).content
file_name=f"{title}.jpg"
with open(file_name,'wb') as file:
    file.write(image)
    print("图片已下载成功！")


 ~~~
